# Лабораторная работа №9-10: Продвинутое ООП на Kotlin

## Описание
Лабораторная работа посвящена изучению продвинутых концепций объектно-ориентированного программирования в Kotlin.

## Структура проекта
Проект организован в следующие пакеты:

actions/ – интерфейсы действий
characters/ – классы персонажей
example/ – примеры для изучения
modules/ – модули космической базы
resources/ – ресурсы и управление ими
Main.kt – точка входа в программу
## Изученные темы
1. Геттеры и сеттеры
   Кастомные аксессоры для контроля доступа и валидации.
   Пример с классом Hero и свойством health с ограничениями.
2. Инкапсуляция
   Использование модификаторов доступа (private, protected, public).
   Пример с классом OutpostWorker, где уровень (level) защищён от внешнего изменения.
3. Data-классы
   Автоматическая генерация toString(), equals(), hashCode(), copy().
   Пример класса OutpostResource для хранения игровых ресурсов.
4. Абстрактные классы
   Создание базовой логики с помощью abstract class.
   Пример абстрактного класса OutpostModule для модулей космической базы.
5. Интерфейсы
   Определение контрактов поведения с помощью interface.
   Пример интерфейса Movable с реализациями в классах Car и Aircraft.
